How to create secure password

four useful tips to create secure password:

  • Make your password long

    Hackers use a lot of way to crack the accounts password, To protect your password from cracking use long Password Minimum 12 Character until tha Password is long The cracking of it is hard.

  • Include Numbers, Symbols, Uppercase and Lowercase

    Randomly mix SYMBOLS and NUMBERS with letters(UPPERCASE and LOWERCASE), for that you can use our Random Password generator.

  • Do Not Reuse Your Password

    Do not use one password for multiple account because if the hacker can hack one of the accounts by crack the account password or hack the site, can easily use your information to get other accounts in another sites you registered in.

  • Do Not Use Personal Information

    Avoid from using personal information to create password because hacker first of all use personal information to crack the passwords.
